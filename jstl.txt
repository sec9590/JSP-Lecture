JSTL
JSP Standard Tag Library
JSP에서 사용 가능한 표준 태그 라이브러리
JSP 코드가 깔끔하고 가독성이 높아짐

--------------------------------------------------------------------------

JSTL 커스텀 태그 5가지

1. 기본기능(Core) : c -> http://java.sun.com/jsp/jstl/core
- 일반 프로그램이 언어에서 제공하는 것과 유사한 변수 선언,
실행 흐름의 제어 기능을 제공


2. 형식화(Format) : fmt -> http://java.sun.com/jsp/jstl/fmt
- 숫자, 날짜, 시간을 포매팅하는 기능과 국제화, 다국어 지원 기능을 제공


3. 데이터베이스(SQL) : sql -> http://java.sun.com/jstl/sql
- 데이터베이스의 데이터를 입력/수정/삭제/조회하는 기능을 제공


4. XML처리(XML) :x -> http://java.sun.com/jstl/xml
- xml 문서를 처리할 때 필요한 기능


5. 함수처리(Funtions) : fn -> http://java.sun.com/jsp/jstl/fn
- 문자열 처리하는 함수를 제공

JSTL core 태그

<c:set> : 변수값 설정
<c:remove> : 변수에 저장된 값 삭제
<c:if> : 조건에 따라 처리를 달리 할 때 사용
<c:choose> : 여러 조건에 따라 처리할 때 사용
<c:forEach> : 반복 처리를 위해 사용
<c:forTokens> : 구분자로 분리된 각각의 토큰을 처리할 때 사용
<c:import> : 외부 자원을 url을 지정하여 가져다 사용
<c:redirect> : 지정한 경로로 이동
<c:url> : url 재작성
<c:out> : 데이터를 출력, <%= =%>를 대체하는 기능
<c:catch> : 예외 처리에 사용

---------------------------------------------------------------------------

<c:forEach> 태그에 사용할 수 있는 속성 중에서 varStatus 속성이 있습니다.
varStatus 속성은 배열이나 컬렉션과 같은 집합체에서 항목의 인텍스 값을 사용해야할 경우에 사용
index = 0부터
count = 1부터

---------------------------------------------------------------------------

동일한 이름의 체크박스가 여러개 존재할 때 html문서에서 여러 항목을 선택하여 전송된 정보를 표현하기 위해서 표현 언어에서는 paramValues를 사용합니다.
paramValues는 지정한 이름을 가진 파라미터의 모든 값을 배열 형태로 얻어오기 때문에 동일한 이름으로 여러번 값이 전송되어도 이를 처리할 수 있다.

JSTL fmt : 국제와 지역화 태그로 jsp페이지에서 다양한 언어를 지원 받을 수 있도록 할 수 있음

1. 숫자 날자 형식
- formatNumber : 숫자를 양식에 맞춰서 출력
- formatDate : 날짜정보를 담고 있는 객체를 포맷팅 하여 출력할 때 사용
- parseDate : 문자열을 날짜로 파싱
- parseNumber : 문자열을 수치로 파싱
- setTimeZone : 시간대별로 시간을 처리할 수 있는 기능을 제공
- timeZone : 시간대별로 시간을 처리 가능

2. 로케일 지정
- setLocale : 나라마다 사용하는 화폐의 종류나 날짜를 표현하는 방식이 다름
- requestEncoding
